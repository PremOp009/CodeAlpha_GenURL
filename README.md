## 📌 Overview

GenURL is a fast and modern URL shortening platform designed to simplify link sharing with a clean UI and seamless user experience.

The project also includes a Chrome Extension that allows users to shorten URLs instantly directly from the browser.

---

## ✨ Features

- 🔗 Instant URL shortening
- 📋 One-click copy to clipboard
- ⚡ Fast and responsive performance
- 🎨 Modern UI with Tailwind CSS
- 📱 Fully responsive design
- 🧩 Chrome Extension support
- 🚀 Cloud deployment with Vercel

## 🧩 Chrome Extension Setup

### Load Extension in Chrome

1. Open Chrome Browser
2. Navigate to:

```txt
chrome://extensions/
```

3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select the `extension` folder

Done ✅
